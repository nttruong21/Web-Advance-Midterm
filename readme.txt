Hướng dẫn chạy project:
B1: Clone project từ github:
B2: Config lại file .env nếu cần.
B3: Nếu thay đổi PORT trong file .env thì phải thay đổi PORT trong dòng 114 (home.hbs).
B4: Mở cửa sổ dòng lệnh terminal và gõ npm i để cài đặt các modules. Tiếp theo gõ npm start để chạy server.
B5: Vào trình duyệt gõ  localhost:[PORT] và thao tác với trang web.
